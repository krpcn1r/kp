## `showTariffs()`
Таблица с 5 колонками: ID / Название / Цена/мес / Скорость / Описание.  
Данные из `tariffs.json` → [[Хранилище данных]].

## `editClients()`
Таблица с inline-редактированием: ФИО, Телефон, Тариф (← →), Баланс, Статус.  
**Del** на выбранной строке — удаление с подтверждением.  
Тариф выбирается из `tariffs.json`, не вводится вручную.
