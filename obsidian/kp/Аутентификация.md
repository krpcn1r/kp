## Компоненты

### [[AuthManager]]
Статический класс. Хранит текущего пользователя в памяти.

```cpp
static int  registerUser(login, password, role)
static bool loginUser(login, password)
static void logout()
static bool isUserLoggedIn()
static User getCurrentUser()
static int  changePassword(currentPassword, newPassword)
```

**Коды ошибок `registerUser`:**
- `0` — успех
- `1` — логин занят
- `2` — слабый пароль
- `3` — пустые поля

**Коды ошибок `changePassword`:**
- `0` — успех
- `1–6` — различные ошибки валидации (длина, спецсимволы, старый пароль)

---

### [[AuthMenu]]
Показывает начальный экран с кнопками «Войти» и «Зарегистрироваться».

```cpp
static bool show()   // возвращает false если пользователь вышел
```

---

### [[Login]]
Форма с двумя полями: логин и пароль. Вызывает `AuthManager::loginUser()`.

---

### [[Register]]
Форма регистрации. Вызывает `AuthManager::registerUser()` с ролью `OPERATOR`.
