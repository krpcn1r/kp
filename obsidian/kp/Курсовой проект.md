# Курсовой проект — АРМ Мобильного Оператора

> Автоматизированное рабочее место оператора связи. Консольное TUI-приложение на C++17 для Windows.

---

## Модули

| Модуль | Назначение |
|--------|-----------|
| [[Аутентификация]] | Вход, регистрация, смена пароля |
| [[Управление клиентами]] | База абонентов, поиск, добавление, удаление |
| [[Администрирование]] | Управление пользователями, бэкапы |
| [[Пользовательский интерфейс]] | Главное меню, просмотр тарифов |
| [[Ядро системы]] | БД, рендер, ввод |
| [[Структуры данных]] | Client, User, Tariff |
| [[Хранилище данных]] | JSON-файлы, бэкапы |

---

## Архитектура

```
main.cpp
  └── AuthMenu::show()
        ├── showLoginScreen()
        └── showRegisterScreen()
              └── HomeMenu::show()
                    ├── ClientMenu::showList()
                    ├── ClientMenu::showSearch()
                    ├── ClientMenu::showAddClient()
                    ├── HomeMenu::editClients()
                    ├── HomeMenu::showTariffs()
                    └── AdminPanel::showAdminPanel()
```

---

## Стек технологий

- **Язык:** C++17
- **Платформа:** Windows (WinAPI консоль)
- **Сериализация:** [nlohmann/json](https://github.com/nlohmann/json)
- **Сборка:** Visual Studio / MSBuild (`kp.sln`)
- **Ввод:** `<conio.h>` — `_getch()`
