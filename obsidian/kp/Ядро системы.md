## [[Database]]

Статический класс. Использует `nlohmann/json`.

```cpp
static bool          saveUsers(users)
static vector<User>  loadUsers()

static bool            saveClients(clients)
static vector<Client>  loadClients()

static vector<Tariff>  loadTariffs()   // только чтение

// Пути к файлам (константы):
static const string USERS_FILE    = "data/users.json"
static const string CLIENTS_FILE  = "data/clients.json"
static const string TARIFFS_FILE  = "data/tariffs.json"
```

Сериализация через макрос `NLOHMANN_DEFINE_TYPE_NON_INTRUSIVE` в структурах.

## [[Render]]

Функции отрисовки консольного UI через WinAPI.

**Примитивы:**
```cpp
void clearScreen()
void setCursor(x, y)          // SetConsoleCursorPosition
void setColor(color)          // SetConsoleTextAttribute
void clearLine(x, y, width, color)
```

**Рамки:**
```cpp
void drawBox(x, y, w, h, color)
void drawDoubleBox(x, y, w, h, color)   // ╔═╗╚═╝
```

**Таблицы:**
```cpp
void drawTableHeader(y, columns, separatorXs, textColor)
void drawTableSeparator(x, y, width, separatorXs, color)
void drawTableCell(x, y, width, value, color)
```

**Ввод:**
```cpp
string processInput(x, y, width, currentInput, isPassword, exitKey&, warningY)
void   drawInputContent(x, y, width, input, isPassword, isActive)
```

**Диалоги:**
```cpp
bool showConfirmation(message)   // ДА / НЕТ диалог
void showPlaceholder(title)      // заглушка экрана
void drawFooter(y, hasBack)      // подсказки снизу
```

## [[InputHandler]]

```cpp
// Коды клавиш (namespace Key):
UP=256, DOWN=257, LEFT=258, RIGHT=259
ENTER=13, TAB=9, ESC=27, BACKSPACE=8, DEL=260

// Методы:
static int  getExtKey()     // _getch() + обработка расширенных кодов (224+scan)
static void waitAnyKey()    // ожидание любой клавиши
```

Расширенные коды (после `224`): `72`=UP, `80`=DOWN, `75`=LEFT, `77`=RIGHT, `83`=DEL.
