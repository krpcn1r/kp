static int  addClient(name, phone, tariff="", balance=0.0)

### [[ClientMenu]]
UI-экраны для работы с абонентами.

```cpp
static void showList()            // таблица всех клиентов с inline-редактированием
static void showSearch()          // форма поиска по 4 полям
static void showAddClient()       // форма добавления нового абонента
static void showEditClient(id)    // заглушка
```

**`showList()`** — таблица с 6 колонками (ID, ФИО, Телефон, Тариф, Баланс, Статус).  
Inline-редактирование: Enter → режим правки строки, Tab/↑↓ — между полями.

**`showSearch()`** — 4 отдельных поля (ID, ФИО, Телефон, Тариф).  
Незаполненные поля игнорируются. Enter на последнем поле — запуск поиска.  
Результаты — карточки клиентов, ↑↓ для листания.

**`showAddClient()`** — поля: ФИО, Телефон, Тариф (↑↓ из списка), Баланс.  
ID назначается автоматически, статус = Активен.

## Навигация по клавишам

| Клавиша | Действие |
|---------|---------|
| ↑ / ↓ | Навигация по строкам |
| Enter | Войти в редактирование |
| Tab | Следующее поле |
| Del | Удалить клиента (с подтверждением) |
| Esc | Отмена / назад |
| ← / → | Смена тарифа (в редактировании) |
| Пробел | Переключить статус |
