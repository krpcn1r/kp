# Структуры данных

> Определения структур (POD-типов), используемых по всему проекту.  
> Все сериализуются через `NLOHMANN_DEFINE_TYPE_NON_INTRUSIVE`.

← [[Курсовой проект]]

---

## User
**Файл:** `user/UserStruct.h`

```cpp
enum class Role { ADMIN, OPERATOR };

struct User {
    std::string login;
    std::string password;   // хранится в открытом виде
    Role        role;
};
```

Используется в: [[Аутентификация]], [[Администрирование]]  
Хранится в: [[Хранилище данных]] → `users.json`

---

## Client
**Файл:** `clients/ClientStruct.h`

```cpp
struct Client {
    int         id;           // автоинкремент
    std::string phoneNumber;  // уникальный
    std::string fullName;
    std::string tariffName;   // имя из tariffs.json
    double      balance;
    bool        isActive;     // false = заблокирован
};
```

Используется в: [[Управление клиентами]], [[Пользовательский интерфейс]]  
Хранится в: [[Хранилище данных]] → `clients.json`

---

## Tariff
**Файл:** `clients/TariffStruct.h`

```cpp
struct Tariff {
    int         id;
    std::string name;
    double      pricePerMonth;
    int         speedMbps;
    std::string description;
};
```

Используется в: [[Управление клиентами]], [[Пользовательский интерфейс]]  
Хранится в: [[Хранилище данных]] → `tariffs.json` (только чтение)

## TableColumn
**Файл:** `core/Render.h`

```cpp
struct TableColumn {
    int         x;
    int         width;
    std::string title;
};
```

Вспомогательная структура для [[Render]] — описывает колонку таблицы.

## HomeResult
**Файл:** `user/HomeMenu.h`

```cpp
enum class HomeResult {
    LOGOUT,     // вернуться в меню авторизации
    EXIT_APP    // закрыть программу
};
```

Возвращается из `HomeMenu::show()` в `main.cpp`.
