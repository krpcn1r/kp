## Файлы данных

### `data/clients.json`
База абонентов. Массив объектов [[Структуры данных#Client|Client]].

```json
[
  {
    "id": 1,
    "phoneNumber": "+7...",
    "fullName": "Иванов Иван",
    "tariffName": "Базовый",
    "balance": 150.00,
    "isActive": true
  }
]
```

Читает/пишет: [[Управление клиентами]] через `ClientManager`  
Редактируется в: [[Пользовательский интерфейс]] через `editClients()`

---

### `data/users.json`
Учётные записи. Массив объектов [[Структуры данных#User|User]].

```json
[
  { "login": "admin", "password": "!!!qwe123", "role": "ADMIN" },
  { "login": "operator1", "password": "!!!qwe123", "role": "OPERATOR" }
]
```

Читает/пишет: [[Аутентификация]] через `AuthManager`  
Редактируется в: [[Администрирование]] через `AdminPanel`

---

### `data/tariffs.json`
Тарифные планы. Массив объектов [[Структуры данных#Tariff|Tariff]].  
**Только чтение** — редактирование тарифов в программе не предусмотрено.

| ID | Название | Цена/мес | Скорость |
|----|---------|---------|---------|
| 1 | Лайт | 149 ₽ | 20 Мбит/с |
| 2 | Базовый | 299 ₽ | 50 Мбит/с |
| 3 | Стандарт | 499 ₽ | 100 Мбит/с |
| 4 | Премиум | 799 ₽ | 300 Мбит/с |
| 5 | Бизнес | 1499 ₽ | 1000 Мбит/с |

Читает: [[Управление клиентами]], [[Пользовательский интерфейс]]

## Бэкапы

**Папка:** `data/backups/YYYY-MM-DD_HH-MM-SS/`

Каждый бэкап содержит копии:
- `clients.json`
- `users.json`
- `tariffs.json`

Создаются и восстанавливаются через [[Администрирование]] → `AdminPanel::createBackup()` / `restoreBackup()`.
