## Компоненты

### [[AdminPanel]]
Статический класс со всеми функциями администрирования.

```cpp
static void showAdminPanel()    // меню с 6 пунктами
static void showUsersList()     // просмотр списка пользователей
static void editUser()          // редактирование: логин, пароль, роль
static void deleteUser()        // удаление с подтверждением
static void createBackup()      // создать бэкап в data/backups/ДАТА/
static void showBackups()       // список существующих бэкапов
static void restoreBackup()     // восстановление из выбранного бэкапа
```

---

## Функциональность

### Управление пользователями
- Таблица пользователей (Логин / Пароль / Роль)
- Inline-редактирование: логин, пароль (с маскировкой `***`), роль (← →)
- Удаление с диалогом подтверждения `showConfirmation()`
- Защита: нельзя удалить последнего администратора

### Система бэкапов
- Бэкап создаётся в `data/backups/YYYY-MM-DD_HH-MM-SS/`
- Копируются: `clients.json`, `users.json`, `tariffs.json`
- Восстановление перезаписывает текущие файлы данных
- Восстановление также требует подтверждения
