# АРМ для мобильного оператора

Автоматизированное рабочее место оператора связи — консольное приложение для Windows с интерактивным TUI-интерфейсом.

## Технологический стек

- **Язык**: C++ (стандарт C++17)
- **Платформа**: Windows (WinAPI — управление цветом, курсором, клавишами)
- **Библиотеки**: `nlohmann/json` (header-only, включена в проект)
- **Сборка**: Visual Studio Solution (`.sln`)

## Структура проекта

```
kp/
├── auth/           — авторизация: вход, регистрация, AuthManager
├── user/           — главное меню оператора (HomeMenu)
├── clients/        — работа с базой абонентов (ClientManager, ClientMenu)
├── admin/          — панель администратора (AdminPanel)
├── core/
│   ├── Render      — UI-движок: рамки, таблицы, поля ввода, футер
│   ├── InputHandler— перехват клавиш (стрелки, Tab, Enter, Esc)
│   └── Database    — чтение/запись JSON (users.json, clients.json)
└── data/           — JSON-файлы базы данных и папка backups/
```

## Функциональность

### Авторизация

- Вход по логину и паролю
- Регистрация с проверкой сложности пароля (8+ символов, цифра, спецсимвол, только латиница)
- Два уровня доступа: **Администратор** и **Оператор**
- Смена пароля из главного меню

### Главное меню (оператор)

| Пункт | Описание |
|---|---|
| Просмотр базы клиентов | Таблица абонентов с inline-редактированием |
| Поиск по базе | Поиск по ФИО, телефону или ID |
| Оформление нового клиента | Форма добавления абонента |
| Сменить пароль | Смена текущего пароля |

Администратор дополнительно видит пункт **Админ-панель**.

### База абонентов (просмотр и редактирование)

Таблица с 6 колонками: **ID · ФИО · Телефон · Тариф · Баланс · Статус**

- `↑` / `↓` / `Tab` — навигация по строкам
- `Enter` на строке — войти в режим редактирования
- В режиме редактирования `Tab` / `↑` / `↓` переключают поля, `Enter` переходит к следующему
- На поле **Статус**: `Space` / `←` / `→` переключают Активен / Заблок., `Enter` сохраняет
- `Esc` — отменить редактирование

Валидация при сохранении: ФИО и телефон не пустые, телефон уникален, баланс — корректное число.

### Панель администратора

#### Управление пользователями

- **Просмотр** — таблица всех учётных записей (логин, пароль скрыт, роль)
- **Редактирование** — inline-изменение логина, пароля и роли прямо в таблице
- **Удаление** — с подтверждением; последнего администратора удалить нельзя

#### Резервные копии БД

- **Создать бэкап** — копирует `users.json` и `clients.json` в `data/backups/YYYY-MM-DD_HH-MM-SS/`
- **Просмотр бэкапов** — список всех резервных копий (новейшие первыми)
- **Восстановление** — выбор бэкапа из списка, подтверждение, копирование файлов обратно

## Навигация (общие горячие клавиши)

| Клавиша | Действие |
|---|---|
| `↑` / `↓` / `Tab` | Навигация между пунктами / строками |
| `Enter` | Выбрать / подтвердить / сохранить |
| `Esc` | Назад / отменить |
| `Space` / `←` / `→` | Переключить булево значение (роль, статус) |

## Хранение данных

Данные хранятся в JSON-файлах в папке `data/`:

- `users.json` — список пользователей системы (логин, пароль, роль)
- `clients.json` — база абонентов (id, телефон, ФИО, тариф, баланс, статус)
- `backups/` — папки с резервными копиями, имя папки = дата и время создания
