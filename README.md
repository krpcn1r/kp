# АРМ для мобильного оператора

## 🛠 Технологический стек

- **Язык**: C++ (стандарт 11/14/17).
- **Платформа**: Windows (использование WinAPI для управления цветом и курсором консоли).
- **Библиотеки**: `nlohmann/json` (включена в проект как header-only).
- **Сборка**: Visual Studio Solution (.sln).

## 📁 Структура проекта

Проект организован по модульному принципу:

- `kp/` — корневая папка с исходным кодом:
  - `auth/` — логика авторизации, `AuthManager` и экраны Входа/Регистрации.
  - `user/` — данные текущего пользователя и главный дашборд (`HomeMenu`).
  - `clients/` — управление базой абонентов (`ClientManager`) и структуры данных.
  - `admin/` — инструменты администратора и системные конфигураторы.
  - `core/` — ядро системы (инструментарий):
    - `Render` — мощный UI-движок (отрисовка рамок, динамический футер, валидация ввода).
    - `InputHandler` — кросс-платформенный (Windows) перехват клавиш управления.
    - `Database` — централизованный доступ к данным (JSON).
  - `data/` — хранилище JSON-файлов базы данных (`users.json`, `clients.json`).
- `main.cpp` — точка входа, реализующая State Machine для переключения между Auth- и Home-режимами.
- `kp.sln` — файл решения для Visual Studio.
