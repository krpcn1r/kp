# Структура проекта АРМ (Автоматизированное рабочее место) на с++(консольное приложение)

## 📂 admin/ (Панель администратора)
* `AdminPanel.cpp` / `.h` — Главное меню и логика прав администратора.
* `ConfigWizard.cpp` / `.h` — Мастер первичной настройки системы.
* `LogViewer.cpp` / `.h` — Модуль просмотра системных логов и событий.

---

## 📂 auth/ (Авторизация и пользователи)
* `AuthManager.cpp` / `.h` — Сервис управления сессией и проверкой прав.
* `Login.cpp` / `.h` — Экран входа в систему.
* `Register.cpp` / `.h` — Экран регистрации новых пользователей.
* `UserStruct.h` — Описание структуры `User` (фиксированные размеры, enum Role).

---

## 📂 clients/ (Учет абонентов/клиентов)
* `Billing.cpp` / `.h` — Финансовый модуль (баланс, тарифы, оплата).
* `ClientManager.cpp` / `.h` — Управление списком клиентов (CRUD операции).
* `ClientStruct.h` — Описание структуры `Client`.

---

## 📂 core/ (Системное ядро)
* `Database.cpp` / `.h` — Низкоуровневая работа с бинарными файлами (`.dat`).
* `InputHandler.cpp` / `.h` — Обработка ввода с клавиатуры.
* `Render.cpp` / `.h` — Движок отрисовки интерфейса (WinAPI, цвета, рамки).
* `Utils.cpp` / `.h` — Вспомогательные функции (валидация, работа с датами).

## 📂 data/ (База данных)
* `settings.dat` - файл с настройками системы.
* `users.dat` — Бинарный файл с данными пользователей.
* `clients.dat` — Бинарный файл с данными клиентов.
